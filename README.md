Este repositório apresenta **simulações controladas e seguras** de comportamentos inspirados em malwares reais, com foco **exclusivamente educacional**.

O objetivo é proporcionar uma compreensão prática de conceitos fundamentais de cibersegurança, sem causar qualquer risco ao sistema do usuário.

---

## 📚 Sobre o Projeto

O projeto aborda dois tipos clássicos de malware amplamente estudados na área de segurança da informação:

- 🔐 **Ransomware Simulado**  
- ⌨️ **Keylogger Simulado**

As implementações foram desenvolvidas de forma **limitada e segura**, garantindo que não executem ações maliciosas reais.

---

## ⚠️ Aviso Importante

> 🚨 Este projeto **NÃO é um malware real**.

- Todo o código foi projetado para rodar em ambiente controlado  
- Nenhuma ação prejudicial é executada  
- Uso exclusivo para **estudo, aprendizado e portfólio**

O uso indevido deste material para fins maliciosos é de total responsabilidade do usuário.

---

## 🎯 Objetivos

- Entender o funcionamento básico de malwares
- Simular comportamentos de forma segura
- Desenvolver pensamento crítico em cibersegurança
- Explorar técnicas de defesa e prevenção
- Construir um portfólio técnico em Python

---

## 🛠️ Tecnologias Utilizadas

- Python 3
- Manipulação de arquivos
- Estruturas de controle e automação
- Conceitos básicos de segurança da informação
